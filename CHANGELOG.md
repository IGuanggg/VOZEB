## v0.9.9 - 2026-07-08

- [新增] 后台接口高级配置支持粘贴上游 cURL、请求 JSON 或返回 JSON 示例，自动识别 Base URL、API Key、模型、接口协议、请求字段模板、结果字段、状态字段和参考素材规则。
- [优化] 生图工作台、视频工作台、画布和素材库上传入口支持从电脑拖拽添加图片、视频或音频素材，并保留原有点击上传、剪贴板上传和生成流程。
- [修复] 后台生成日志的视频模型显示去掉内部渠道前缀，避免把 `渠道ID::模型名` 显示成模型名称。
- [调整] 后台接口添加、模型拉取、接口测试和默认模型配置独立为“接口配置”页面，系统设置页只保留账号、邮箱、额度、生成默认值和存储相关配置。
- [优化] 后台新增/编辑接口改为默认只填写接口名称、Base URL、API Key 和一键检测接口，高级配置默认收起，避免新手配置时被复杂字段干扰。
- [优化] 接口一键检测会自动补齐模型列表，并回填文本模型、图片模型、视频模型、接口协议、创建路径、查询路径、结果字段、状态字段、参考素材规则和支持时长。
- [优化] 接口配置页按检测结果显示文本、生图、图生图、视频、图生视频、参考视频/音频的简单结论。
- [修复] 青衍 `api2.qingyanzhiying.top` 的图生图和图生视频参考图按文档优先提交公网图片 URL；本地参考图只在部署公网域名后转临时公开地址，避免把 base64 直接提交给上游。
- [修复] 后台接口健康检测和系统代理视频类型识别补齐 GlobalAiOpc `/videos/videos` 路径，避免视频接口检测不准或视频请求未按视频类型处理。
- [修复] 兼容 GlobalAiOpc Videos 接口，视频兼容分支新增 `/videos/videos` 创建任务和 `/result/{id}` 查询结果，避免上游已完成但前端一直停留在生成中。
- [修复] GlobalAiOpc 视频请求按文档发送 `duration`、`ratio`、`resolution`、`referenceImages`、`referenceVideos`、`referenceAudios` 等字段，避免旧兼容字段和新字段混用造成参数冲突。
- [修复] GlobalAiOpc 图生视频参考素材只提交公网 URL；本地 `blob` 或不可访问素材会直接提示，避免上游扣费后按纯文生视频生成。
- [修复] URL 型视频接口的本地参考图会自动转存为临时公开地址，再作为 `referenceImages` 提交给上游；未配置公网域名时继续拦截并提示。
- [调整] GlobalAiOpc 视频时长按文档限制为 4-15 秒，不再套用旧兼容接口的 5/10/15 秒档位。
- [修复] 接口高级配置的协议、创建路径和查询路径进入实际视频生成链路，避免一键检测保存后生成仍按旧兼容路径请求。
- [修复] 接口高级配置的参考素材规则进入图片任务运行时，图生图可按接口说明选择 multipart 文件上传、JSON 参考图或公网参考图 URL。
- [优化] 兼容视频接口支持使用高级配置的请求字段模板，模板里的模型、提示词、时长、比例、清晰度和参考素材会按当前生成参数自动替换。
- [修复] 兼容视频接口优先按高级配置的结果字段和状态字段读取上游响应，避免自定义接口已完成但前端仍停留在生成中。
- [修复] 图片工作台图生图恢复 OpenAI 兼容接口的 multipart 参考图上传；青衍接口继续使用公网参考图 URL，避免不同接口的参考图提交方式互相影响。
- [修复] code2alita 等 JSON 图生图接口不再被后台通用参考图规则误判为 multipart，图生图固定按 `images[].image_url` 提交；后台一键检测也会回填正确的参考素材规则。
- [优化] 后台高级配置新增 `sub2api` 协议选项，检测到 sub2api 接口后会保存该协议，避免换域名后图生图参考图提交方式再次被误判。
- [修复] 上游返回 `backend:6666` 等内部媒体地址时按当前上游域名重写同一路径，避免前端代理 404 或一直停留在生成中。
- [调整] JSON 图生图仅在参数格式错误时切换请求体，遇到 5xx/524 不再自动换请求体重发，减少重复扣费风险。

## v0.9.8 - 2026-07-07

- [修复] 画布移动端连线与双指缩放交互，手机端可以正常连接节点并缩放画布。
- [修复] 生图工作台失败记录刷新后重复出现的问题，保留原有文生图、图生图和上传参考图生图流程。
- [修复] 生图工作台获取上游返回图的地址解析与展示逻辑，避免生成成功后前端一直显示生成中或出现本地错误地址。
- [修复] 图生图参考图提交方式，保留图片工作台与视频工作台不同的参考图上传路径，避免互相影响。
- [修复] 图生视频任务创建、轮询和结果回填流程，支持上游视频任务完成后正常拿到视频结果。
- [修复] 视频时长配置传递异常，避免前端选择 5 秒但上游生成 10 秒。
- [修复] 画布生成图片和生成视频入口的任务提交与结果回填，保持画布节点生成流程可用。
- [修复] 图片工作台同一记录再次生成时，结果继续追加到右侧同一记录中，不再新建记录。
- [修复] 公共提示词按标签和分类切换后内容不更新的问题。
- [调整] 公告页面《返回首页》移动到标题右侧，减少页面顶部空间占用。
- [调整] 视频工作台补充原作者音频设置、语速和音频格式入口，并改为更省空间的选择式控件，避免影响原有视频生成功能。
